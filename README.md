# Installation
Instructions:
1. (In Claude) Go to "Customize"
2. "Create new skills"
3. Click "+" (Add skill)
4. "Create Skill"
5. "Upload a skill"
6. Drag & Drop or Click to Upload, then select the .zip file with the agent skill you wish to use
    * You CAN do bulk upload, and Claue will automatically differentiate the different .zip files into different skills
